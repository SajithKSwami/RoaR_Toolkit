# RoaR Companion Toolkit

**A practical operating system for becoming known, findable, and steady inside the hiring machine.**

This is the free companion toolkit to the book *Rumblings of a Recruiter* by [Sajithkumar Swaminathan](https://www.sajithkumarswaminathan.com) — recruiter, career strategist, and career guide.

The book names what the hiring machine does. **This toolkit helps you decide what to do next.**

## What's inside

It's built on the **ROARS** method — five steps, in order:

| Step | Name | What it does |
|---|---|---|
| **R** | Read the Machine | Spot ghost postings and dead roles before you waste energy on them. |
| **O** | Own Your Signal | Get clear on what you're known for — resume, LinkedIn, and proof. |
| **A** | Activate Relationships | Build the right conversations before the posting gets crowded. |
| **R** | Reduce the Damage | Protect your energy, identity, and judgment in a long search. |
| **S** | Ship the Search | Track signal (not effort), measure conversion, adjust weekly. |

Plus a one-page **Quick Card**, an **Interview Toolkit**, a **Negotiation Toolkit**, and a **30-Day Search Sprint**.

*RoaR* is the book and the brand. *ROARS* is the method.

---

## Who it's for

Anyone running a real job search who is tired of applying into silence — early-career to senior, across functions. If you've ever sent 50 applications and heard nothing back, this is the system that turns that effort into signal.

---

## How to start

Don't try to use everything. Pick **one tool** this week:

- **New to the search?** → *Candidate Signal Statement* + *20-Person Visibility Map*
- **Deep in a long search and drained?** → *Search Energy Budget* + *Stop-Time Rule*

One tool used beats ten tools admired.
